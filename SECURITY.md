# Security Policy

## Supported Versions

| Version | Supported |
|---|---|
| 2.x.x | ✅ |
| 1.x.x | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly through GitHub's private vulnerability reporting:

1. Go to the [Security tab](https://github.com/lopatnov/translate/security) of this repository
2. Click **"Report a vulnerability"**
3. Fill in the details of the vulnerability

Please do **not** open a public issue for security vulnerabilities.
